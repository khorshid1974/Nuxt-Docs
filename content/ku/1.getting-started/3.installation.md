## ئامرازی `create-docus`

::steps
### دروستکردنی فۆڵدەری بەڵگەنامە

ئامرازی `create-docus` بەکاربهێنە بۆ دروستکردنی پڕۆژەیەکی نوێی Docus:

```bash [Terminal]
npx create-docus my-docs
```

دەتوانیت لە نێوان دوو تێمپلێت هەڵبژێریت:

- `default`: ڕێکخستنی بنەڕەتی Docus بۆ بەڵگەنامەی یەک زمانی
- `i18n`: پشتگیری فرەزمانی لەخۆدەگرێت بۆ بەڵگەنامەی فرەزمان

```bash [Terminal]
# Create with i18n template
npx create-docus my-docs -t i18n
```

پێشنیار دەکەین بەکارهێنانی بەڕێوەبەری پاکێجی `npm`.

### دەستپێکردنی سێرڤەری بەڵگەنامە لە دۆخی پەرەپێدان

برۆ ناو فۆڵدەری بەڵگەنامەکەت و سێرڤەری بەڵگەنامە دەستپێبکە لە دۆخی پەرەپێدان:

```bash [Terminal]
cd my-docs
npm run dev
```

پێشبینینی ناوخۆیی بەڵگەنامەکەت بەردەست دەبێت لە <http://localhost:3000>

### نووسینی بەڵگەنامەکەت

سەردانی بەشی [سینتاکسی Markdown](/essentials/markdown-syntax) بکە بۆ فێربوونی چۆنیەتی نووسینی بەڵگەنامەکەت.
::

## Layer Integration

Docus v4 uses a **Nuxt layer-based approach**, you can extend the Docus layer directly in your `nuxt.config.ts` with `extends: ['docus']`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  extends: ['docus']
})
```
