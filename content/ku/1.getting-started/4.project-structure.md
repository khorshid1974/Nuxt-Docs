## پێکهاتەی گشتی

بریتییە لە **لێیەرێکی Nuxt** کە ئەپڵیکەیشنی ستانداردی Nuxt فراوان دەکات بە تایبەتمەندییەکانی بەڵگەنامە. ئەمە نەرمی پڕۆژەیەکی تەواوی Nuxt دەبەخشێت.

سیستەمی بۆلۆنا سیستەمەکەی تابیبەتە بە بەروەبردنی زنکۆکان وئاسانکری بۆ جێ بە چو کردنی بۆلۆنا دەکات

```bash
my-docs/
├── content/             # ناوەڕۆکی Markdown ی تۆ
│   ├── index.md         # پەڕەی سەرەکی
│   └── docs/            # پەڕەکانی بەڵگەنامە
├── public/              # ئاسەتە جێگیرەکان
└── package.json         # پێداویستییەکان و سکریپتەکان
```

هێشتا دەتوانیت هەر تایبەتمەندی یان فایلێکی پڕۆژەیەکی کلاسیکی Nuxt بەکاربهێنیت:

```bash
my-docs/
├── nuxt.config.ts       # ڕێکخستنی Nuxt (زیادکردنی مۆدیوڵی زیادە، پێکهاتەکان، هتد)
├── app/                 # فۆڵدەری ئەپ
    ├── app.config.ts    # ڕێکخستنی ئەپ
│   ├── components/      # پێکهاتەکان (زیادکردنی پێکهاتەکانی خۆت)
│   ├── layouts/         # دیزاینەکان (زیادکردنی دیزاینەکانی خۆت)
│   └── pages/           # پەڕەکان (زیادکردنی پەڕەکانی خۆت)
└── server/              # کۆدی سێرڤەر (زیادکردنی کۆدی سێرڤەری خۆت)
```

### فۆڵدەری `content/`

لێرەدا پەڕەکان دەنووسیت بە Markdown. Docus بە شێوەیەکی خودکار ڕێڕەوەکان دروست دەکات بەپێی پێکهاتەی فایلەکەت.

**پێکهاتەی یەک زمانی:**

```bash
content/
├── index.md
├── getting-started.md
└── guide/
    ├── introduction.md
    └── configuration.md
```

**Multi-language structure (with i18n):**

```bash
content/
├── en/
│   ├── index.md
│   └── guide/
│       └── introduction.md
└── fr/
    ├── index.md
    └── guide/
        └── introduction.md
```
