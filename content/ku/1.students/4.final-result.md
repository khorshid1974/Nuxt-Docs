ئەم پەڕەیە قوتابیان و مامۆستایان ئەتوانن ئەداى ئەکادیمى لە هەموو سمستەرە تەواوبووەکان ببینن لەکاتی راگەیاندنی ئەنجامەکان لە لایەن لێژنەی بالای تاقیکردنەوەکان

## ١. سەرووى زانیارییەکانى قوتابى

هەر پەڕەى ئەنجامێک لە وردەکارییەکانى ناسنامەى قوتابى دەستپێدەکات بۆ دڵنیاکردنەوەى ئەوەى تۆمارەکان بە ئاسانى دەناسرێنەوە.

- **ناوى قوتابى (Student Name):** بۆ نموونە، سیما رئووف محمد شەریف
- **کۆدى قوتابى (Student Code):** بۆ نموونە، ETAC03M220338
- **سمستەر (Semester):** تەرمى ئەکادیمى تایبەتەکە دیار دەکات (١ تا ٨).

![سەرووى زانیارییەکانى قوتابى]()

## ٢. خشتەى ئەداى ئەکادیمى

دەرەنجامى بەڵگەنامەکە خشتەیەکى وردەکارانەیە کە نمرەى قوتابى بۆ هەر بابەتێک لە سمستەرێکى دیارى پیشان دەدات.

| بابەت (Subject) | نمرە (Grade) | ئاست (Level) |
|---|---|---|
| بەڕێوەبردنى بانک (إدارة المصارف) | ٦٣.٤ | D |
| بەڕێوەبردنى بەرهەمهێنان و کارپێکردن | ٧٧.٦٦ | +C |
| بەڕێوەبردنى ستراتیجى | ٧٨.٧٥ | +C |
| بەڕێوەبردنى ماڵپەڕ | ٧٤.٥ | C |
| بنچینەکانى لێکۆڵینەوەى زانستى | ٦٠.٠ | -D |

![خشتەى ئەداى ئەکادیمى]()

## ٣. کورتەى سمستەر

لە خوارەوەى ڕاپۆرتى هەر سمستەرێک، سیستەمەکە دۆخى کۆتایى و کۆى کرێدیتەکان دەدات:

- **ئەنجامى کۆتایی (Final Result):** دۆخەکە پیشان دەدات وەک "پیرۆزە دەرچووى (Passed)" یان "دەرنەچوو (Failed)".
- **کۆى گشتى کرێدیت (Total Credits):** کۆى کرێدیتەکانى ECTS ى بەدەستهاتوو لە سمستەرەکە (بۆ نموونە، ٢٠.٧٥).

![کورتەى سمستەر]()

## ٤. پێوەرى پلەبەندى نمرە

سیستەمەکە پێوەرى پلەبەندى یەکسانکراو بەکاردەهێنێت بۆ دابەشکردنى نمرەى ژمارەیى بە ئاستەکان:

| ماوەى نمرە | ئاست | ماوەى نمرە | ئاست |
|---|---|---|---|
| ٠ - ٥٩ | F | ٨٠ - ٨٢ | -B |
| ٦٠ - ٦٢ | -D | ٨٣ - ٨٦ | B |
| ٦٣ - ٦٦ | D | ٨٧ - ٨٩ | +B |
| ٦٧ - ٦٩ | +D | ٩٠ - ٩٢ | -A |
| ٧٠ - ٧٢ | -C | ٩٣ - ٩٦ | A |
| ٧٣ - ٧٦ | C | ٩٧ - ١٠٠ | +A |
| ٧٧ - ٧٩ | +C | | |

## ٥. ئاگادارکردنەوەى گرنگ

سیستەمەکە ئاگادارکردنەوەیەکى پێویستى تێدایە دەربارەى بەکارهێنانى ئەم تۆمارە دیجیتاڵییانە:

> **ئاگادارکردنەوە:** ئەم بەڵگەنامەیە تەنها بۆ زانینى ئەنجامەکان بەکاردێت ، بۆ کارى فەرمی بەکار نایەت (This document is for informational purposes only to view results. It is not for official use).
