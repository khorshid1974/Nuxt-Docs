## أداة `create-docus`

::steps
### أنشئ مجلد التوثيق الخاص بك

لكي تستعمل نظام بولونا الخاص بجامعة اربيل التقنية فآنت تختاج الى حساب يتعمد على البريد الالكتروني للجامعة

```bash [Terminal]
npx create-docus my-docs
```

يمكنك الاختيار بين قالبين:

- `default`: إعداد Docus الأساسي لتوثيق بلغة واحدة
- `i18n`: يتضمن دعم التدويل لتوثيق متعدد اللغات

```bash [Terminal]
# Create with i18n template
npx create-docus my-docs -t i18n
```

نوصي باستخدام مدير الحزم `npm`.

### ابدأ خادم التوثيق في وضع التطوير

انتقل إلى مجلد التوثيق وابدأ خادم التوثيق في وضع التطوير:

```bash [Terminal]
cd my-docs
npm run dev
```

ستتوفر معاينة محلية لتوثيقك على <http://localhost:3000>

### اكتب توثيقك

توجه إلى قسم [تنسيق Markdown](/essentials/markdown-syntax) لتعلم كيفية كتابة توثيقك.
::

## Layer Integration

Docus v4 uses a **Nuxt layer-based approach**, you can extend the Docus layer directly in your `nuxt.config.ts` with `extends: ['docus']`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  extends: ['docus']
})
```
