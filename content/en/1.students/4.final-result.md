This page allows students and faculty to view academic performance across all completed semesters.

## 1. Student Information Header

Every result sheet begins with the student's identification details to ensure records are easily identifiable.

- **Student Name (ناوى قوتابی):** e.g., Sima Raouf Mohammed Sharif
- **Student Code (کۆد):** e.g., ETAC03M220338
- **Semester (سمستەر):** Indicates the specific academic term (1 through 8).

![Student Information Header]()

## 2. Academic Performance Table

The core of the document is a detailed table showing the student's grades for each subject within a given semester.

| Subject (بابەت) | Grade (نمرە) | Level (ئاست) |
|---|---|---|
| Bank Management (إدارة المصارف) | 63.4 | D |
| Production & Operations Management | 77.66 | +C |
| Strategic Management | 78.75 | +C |
| E-Site Management | 74.5 | C |
| Scientific Research Principles | 60.0 | -D |

![Academic Performance Table]()

## 3. Semester Summary

At the bottom of each semester's report, the system provides the final standing and credit totals:

- **Final Result (ئەنجامى کۆتایی):** Shows status such as "Passed" (پیرۆزە دەرچووى) or "Failed" (دەرنەچوو).
- **Total Credits (کۆى گشتى کرێدیت):** The total ECTS credits earned during the semester (e.g., 20.75).

![Semester Summary]()

## 4. Grading Scale Reference

The system utilizes a standardized grading scale to categorize numerical scores into levels:

| Score Range | Level | Score Range | Level |
|---|---|---|---|
| 0 - 59 | F | 80 - 82 | -B |
| 60 - 62 | -D | 83 - 86 | B |
| 63 - 66 | D | 87 - 89 | +B |
| 67 - 69 | +D | 90 - 92 | -A |
| 70 - 72 | -C | 93 - 96 | A |
| 73 - 76 | C | 97 - 100 | +A |
| 77 - 79 | +C | | |
